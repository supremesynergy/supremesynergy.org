# supremesynergy.org

Public landing site for Supreme Synergy.

## Stack
- Static HTML/CSS (no build step)
- Deployed via Vercel, auto-deploys from `main`
- DNS: Namecheap → Vercel

## Local preview
Just open `index.html` in a browser, or:
```
npx serve .
```

## Deploy
Pushes to `main` auto-deploy via the Vercel GitHub integration.
